Phishing Attacks



What is Phishing?

Phishing is a type of social engineering attack used by attackers to steal sensitive data of users, such as login credentials and credit card numbers. Actually, these attacks are very common because they are easy to perform. if you go through the internet you can see a lot of tools and applications there in order to perform those. In these attacks, the attacker is misleading the user and intend to perform actions that they really don't want to do. based on the type of attacks, we can identify some types of categories, I would like to mention four of them as Spear phishing, Whaling, Vishing, Email phishing. All the attack types expectation is same, that is stealing the sensitive information, those are different each one of them according to the method they used to send the malicious link, some phishing attacks are targetting persons, some are sharing the malicious link using SMS or via email, that's how they differ each one.

...




How is this happen?

when we analyse those we can identify all the methods are using the same method which is sending a malicious link to the user via email or SMS or anyhow. the task of those attacks is misleading the user and redirect to another page. as an example imagine we get an email, its looks like a message from Facebook and they said “your account has a security error, log in with this link and fix it”, when you click the link it redirect to page that looks like a Facebook login page, apparently there is no any doubt because it really likes the Facebook login page but it can be a phishing page, so you enter your login credentials and attacker stealing your login credentials since the page is fake. Someone can steal the bank account details following the same method, it is also an attacker misleading the user with a fake login page.

protect ourselves against phishing
